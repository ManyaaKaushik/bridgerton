## ‚ùì FAQs

**Q: Can the recipient edit the card?**
A: No! They only see the beautiful finished version. This prevents accidental changes.

**Q: Will the card work if they share the link with someone else?**
A: Yes! Anyone with the link can view the card.

**Q: How long are the links valid?**
A: Links work as long as your browser's storage remains intact. They don't expire.

**Q: Can I edit a card after creating it?**
A: Yes! Until you click "Get Private Link", you can go back and edit. After sharing, create a new card.

**Q: What if I clear my browser data?**
A: Previously created cards will be lost. Always save/bookmark the shareable links you want to keep!

**Q: Can I add more than two names?**
A: The app is designed for sender + recipient. For variations, you can create multiple cards.

**Q: Is my data private?**
A: Yes! Everything is stored on your device only. No servers track your data.

**Q: Can I download the image?**
A: You can print to PDF or take a screenshot to save the card image.

## üõ†Ô∏è Troubleshooting

**The card doesn't show my names**
- Refresh the page
- Check browser console (F12) for errors
- Try a different browser

**Copy button not working**
- Check if page has clipboard permissions
- Try on a secure connection (HTTPS)
- Use manual copy (Ctrl+C) if automatic copy fails

**Form won't submit**
- Ensure both name fields have text
- Try resetting the browser cache
- Refresh and try again

**Links share button not opening**
- WhatsApp: Ensure WhatsApp is installed
- Email: Ensure email client is configured
- Copy link: Should always work as fallback

**Mobile display issues**
- Double-check responsive mode in your browser
- Test on actual mobile device
- Try different mobile browsers
